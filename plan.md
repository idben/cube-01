# 立方塊算數練習

## 程式需求
* 使用 HTML/JS/CSS，三個檔分離
* 不使用 JS 框架，像是 Vue 或是 React 那些
* 使用 Three.js
* JS 中若要抓取元素，使用 quertSelector 或 querySelectorAll，不使用 getElement 系列語法
* 使用 dialog 代替 alert 或 confirm，如果有用到的話
* 使用 Bootstrap 的按鈕設計與顏色設計，但不直接把 Bootstrap 引用進來，如果有用到的話

## 操作需求1
* 畫面中間有隨機生成的立方塊堆疊結構 (6-15 個立方塊)
* 使用正交相機 (無透視效果)
* Y 軸固定，可以以 Y 軸為中心旋轉
* 立方塊固定配色系統:
  - 上面 (+Y): 白色
  - 右面 (+X): 淺灰色
  - 左面 (-X): 深灰色
  - 其他面: 中灰色
* 立方塊之間無間隙，有黑色邊框
* 重力模擬: 無懸空立方塊

## 出題的方向改為：
* 角度改固定
* 先決定最底層為多少個，隨機 4~9
* 由最離視覺最遠的那一個角落開始填方塊
* 再決定第二層方塊數量，隨機數量，但小於等於底層
* 最後再決定第三層，隨機數量，但小於等於第二層
